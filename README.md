Lesson Learned

Melalui repository ini, saya mendokumentasikan langkah awal saya dalam mempelajari mobile development menggunakan Flutter dari nol dengan bantuan berbagai sumber pembelajaran daring. mayoritas waktu saya dipakai untuk menonton video pengenalan dari channel YouTube Erico Darmawan Handoyo sebelum mulai membuat aplikasi To-Do. Dari proses instalasi Flutter hingga memahami dasar sintaks dan logika di dalamnya, materi tersebut membantu saya membangun fondasi yang cukup sebelum mulai mengembangkan aplikasi sendiri.

Selama pengembangan, saya menggunakan AI sebagai alat bantu untuk belajar dan pengerjaan. Namun, saya tidak sekadar menerima hasil prompt secara instan. Saya secara konsisten memantau, memahami, dan mempelajari alasan teknis di balik setiap perubahan. Bagi saya, penggunaan AI telah banyak membantu dalam proses belajar Mobdev.

Sebagai mahasiswa semester 2 Saya memulai pembelajaran Flutter benar-benar dari nol, tanpa pengalaman sebelumnya dalam framework mobile development atau framework of any kind. Prosesnya dimulai dari memahami bagaimana Flutter bekerja secara konsep, seperti perbedaan antara widget stateless dan stateful, bagaimana sistem widget tree tersusun, serta bagaimana mekanisme rebuild terjadi saat state berubah. Saya juga mempelajari dasar sintaks Dart dan bagaimana struktur kode Flutter dibangun. Saya berusaha memahami alasan di balik setiap baris kode, mengapa sebuah widget diletakkan di dalam widget lain, bagaimana context bekerja, serta bagaimana data mengalir dalam aplikasi.
